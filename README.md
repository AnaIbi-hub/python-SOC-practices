# python-SOC-practices
Este repositorio contiene prácticas realizadas durante el curso de Programación en Python de CIFO La Violeta (junio - julio 2025)

Incluye ejercicios sobre:
* estructuras de control (bucles `for` y `while`)
* funciones
* manipulación de cadenas y listas
* list comprehensions
* expresiones regulares (regex)
* limpieza básica de datos
* primeros pasos en interfaces gráficas
